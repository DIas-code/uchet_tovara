# Инструкция: как пользоваться «Учётом товара»

Простое пошаговое руководство для продавца и администратора бутика.
Адрес: **https://uchet-tovara-e1u1.vercel.app**

---

## 1. Вход

1. Открыть адрес приложения.
2. Ввести **почту** и **пароль** своего бутика.
3. Нажать **«Войти»**.

После входа сверху видно название бутика и вашу роль (Администратор/Продавец).
Разделы — в шапке: **Товары, Категории, Отчёты**.

---

## 2. Сначала заведите категории (по желанию)

Категории помогают раскладывать товар по группам (Платья, Юбки, Футболки…).

1. Шапка → **Категории**.
2. В поле сверху ввести название, например `Юбки`, нажать **«Добавить»**.
3. Категорию можно **переименовать** (карандаш) или **удалить** (корзина).

> Категории свои у каждого бутика. Управляет ими администратор.

---

## 3. Добавить товар

1. Шапка → **Товары** → кнопка **«Добавить товар»**.
2. Заполнить:
   - **Код** — обязателен, уникален (например `PLT-001`).
   - **Наименование** — по желанию.
   - **Категория** — выбрать из списка (или «Без категории»).
   - **Цена продажи** — обязательна.
   - **Цена закупки** — по желанию.
   - **Начальный остаток** — сколько штук уже есть (заведётся как «приход»).
3. Нажать **«Сохранить»**. Товар появится в списке.

---

## 4. Контроль количества: приход, продажа, возврат ⭐

Это главное. Остаток меняется **только** через операции — так он всегда точный.

1. На странице **«Товары»** нажать на **код товара** (синеватый текст) —
   откроется **карточка товара**.
2. Нажать кнопку **«Провести операцию»**.
3. Выбрать тип:
   - **Приход** — привезли/докупили товар (остаток увеличится).
   - **Продажа** — продали (остаток уменьшится).
   - **Возврат** — покупатель вернул (остаток увеличится).
4. Указать **количество** и при необходимости **дату** (по умолчанию — сегодня).
5. Нажать **«Провести операцию»**.

Остаток пересчитается сразу. Ниже на карточке — **история всех операций**
по этому товару.

> Продать больше, чем есть на складе, система не даст — покажет ошибку
> «Недостаточно товара на складе», остаток не изменится.

---

## 5. Найти товар и посмотреть по категориям

На странице **«Товары»**:
- **Поиск** — по коду или названию.
- **«Только в наличии»** — скрыть товары с нулевым остатком.
- **Плашки категорий** сверху — нажать «Платья», чтобы видеть только их;
  «Все» — показать всё, сгруппированное по категориям.

---

## 6. Загрузить много товаров сразу (импорт из CSV)

Чтобы не вводить товары по одному, можно загрузить список файлом (админ).

1. Подготовить файл в Excel с колонками (первая строка — заголовки):
   **Код; Наименование; Категория; Цена продажи; Цена закупки; Остаток**
2. Сохранить как **CSV (разделитель — точка с запятой)**.
   > Проще всего: на странице «Отчёты» нажать «Экспорт товаров» — получите файл
   > нужного формата, его можно отредактировать и загрузить обратно.
3. Страница **«Товары»** → кнопка **«Импорт»** → выбрать файл.
4. Появится отчёт: сколько добавлено, сколько пропущено (коды, что уже есть),
   и ошибки по строкам, если были.

Новые категории из файла создаются автоматически. «Остаток» из файла заводится
как приход.

## 7. Отчёты и выгрузка в Excel

1. Шапка → **Отчёты**.
2. Выбрать период (**С… По…**) и нажать **«Показать»**.
3. Сверху — сводка: продано, выручка, приход, возврат.
4. Кнопки **«Экспорт товаров»** и **«Экспорт операций»** — скачивают файл CSV
   (открывается в Excel).

---

## 8. Роли

- **Администратор** — может всё: товары, цены, удаление, категории.
- **Продавец** — приход/продажа/возврат и просмотр; не может удалять товары,
  менять цены и категории.

---

## Частые вопросы

**Не могу удалить товар.** Удаление возможно только при остатке 0. Сначала
спишите остаток (например, операцией «продажа»), потом удаляйте.

**Ошибка при добавлении товара «код уже существует».** В вашем бутике уже есть
товар с таким кодом — коды не повторяются.

**Выкинуло на страницу входа.** Сессия истекла, просто войдите заново.
